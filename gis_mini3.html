When adding data to a GIS, we often refer to them as "layers."

Common Vector Formats:

  1. Most often: Shapefile (.shp)
  2. kml/kmz
  3. GeoJson

Common Raster Formats:

  1. Usually GeoTiff
  2. Also ArcGRID, jpeg2000, others

Spatially enabled databases

(Can store multiple layers/data sets;
both vector & raster data)


  1. Geodatabase (.gdb) [proprietary]
  2. GeoPackage (.gpkg) [open source]

Great for managing projects

Webby Formats

  1. Web Sap Service (WMS); basically an image
  2. Web Feature Service (WFS)
  3. Tiles
